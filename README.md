# 🎮 Tic Tac Toe (Python)

A console-based Tic Tac Toe game built using Python.

## 📌 Features

- Two-player gameplay
- Winner detection
- Draw detection
- Input validation
- Simple console interface

## 🛠 Technologies Used

- Python 3

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/VishuTyagi03/tic-tac-toe-python.git
```

Go into the project folder

```bash
cd tic-tac-toe-python
```

Run

```bash
python TicTacToe.py
```

## 📷 Output

The game is played in the terminal where players take turns entering positions from 1 to 9.
